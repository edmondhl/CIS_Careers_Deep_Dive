# CIS Careers Deep Dive

## Description
Data analyssis project that explores tech employement, salary trends, and hot technologies using government provided data. This project aggregates SOC 15-XXXX tech occupations and generates visualizes for state level insights.

---

## Installation
> pip install -r requirements.txt

## Run
> python main.py

## Sources
* U.S Bureau of Labor Statistics (BLS)
* Occupational Employment and Wage Statistics (OEWS)
* O*Net
